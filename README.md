# 🌌 Skylanders API

¡Bienvenido a **Skylanders API**! Este proyecto es una base de datos en desarrollo diseñada para recopilar información detallada sobre todos los personajes del universo Skylanders y sus variantes jugables.

---

## 🚀 Estado del Proyecto: En Construcción 🏗️

Actualmente, la API se encuentra en fase de desarrollo activo, pero ya cuenta con un avance significativo:
* **Completados:** Todos los personajes y variantes de los 3 primeros juegos (*Spyro's Adventure*, *Giants* y *SWAP Force*).
* **Base de datos:** Soportada y consumida a través de **Supabase**.
* **Filtros y Buscadores:** Ya incluye sistemas de búsqueda separados (actualmente en fase de pruebas).

> ⚠️ **Nota importante:** Por el momento, la API **no es de uso público**, pero estará disponible para la comunidad en próximas actualizaciones.

---

## 🎯 Alcance del Proyecto

Para mantener el enfoque en las figuras principales, la API **solo incluye**:
* ✅ Todos los personajes principales (Skylanders).
* ✅ Variantes jugables (In-game variants como Legendary, Nitro, Dark, etc.).

**Exclusiones actuales:**
* ❌ Objetos mágicos y packs de expansión.
* ❌ Trampas (Traps) y Cristales de Creación.
* ❌ Vehículos.
* ❌ *Chase variants* (variantes físicas de colección que no cambian el aspecto dentro del juego).

---

## 🎨 Características principales

* **Estilo Limpio y Sencillo:** Diseñado con un enfoque básico, intuitivo y fácil de usar tanto para desarrolladores como para fans.
* **Buscadores Avanzados:** Filtros específicos para encontrar a tus Skylanders favoritos por juego, elemento o tipo (en pruebas).
* **Backend Potente:** Gestión de datos eficiente gracias a la infraestructura de Supabase.

---

## 🛠️ Tecnologías utilizadas

* **Base de Datos:** [Supabase](https://supabase.com/)
* *HTML, css y JavaScript*
